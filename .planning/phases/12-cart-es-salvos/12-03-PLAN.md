  <action>
    Modificar SettingsScreen.tsx — NÃO reescrever do zero. Adicionar estado e lógica de cartões salvos sem quebrar as seções existentes.

    Estado novo a adicionar:
    - savedCards: SavedCard[] — inicializa como []
    - loadingCards: boolean — inicializa como true
    - cardError: string | null — inicializa como null
    - removingCardId: string | null — inicializa como null
    - cardToRemove: SavedCard | null — inicializa como null (usado para o dialog)
    - showRemoveDialog: boolean — inicializa como false

    useEffect adicional (na montagem, junto aos demais fetchs):
    apiFetch('/users/me/cards')
      .then(r => r.ok ? r.json() : Promise.reject())
      .then((cards: SavedCard[]) => setSavedCards(cards))
      .catch(() => setCardError('Não foi possível carregar seus cartões. Tente novamente.'))
      .finally(() => setLoadingCards(false))

    Handler handleSetDefault(cardId: string):
    - Chamar apiFetch(`/users/me/cards/${cardId}`, { method: 'PATCH', body: JSON.stringify({ isDefault: true }), headers: { 'Content-Type': 'application/json' } })
    - Se ok: atualizar savedCards localmente (isDefault true para cardId, false para demais), exibir toast "Cartão padrão atualizado."
    - Se erro: toast "Algo deu errado. Tente novamente."

    Handler handleRemoveConfirm():
    - Se !cardToRemove return
    - setRemovingCardId(cardToRemove.id)
    - Chamar apiFetch(`/users/me/cards/${cardToRemove.id}`, { method: 'DELETE' })
    - Se ok (status 204): atualizar savedCards filtrando o removido, toast "Cartão removido.", setShowRemoveDialog(false), setCardToRemove(null)
    - Se erro: toast "Algo deu errado. Tente novamente."
    - finally: setRemovingCardId(null)

    Handler handleRemovePress(card: SavedCard):
    - setCardToRemove(card)
    - setShowRemoveDialog(true)

    Nova seção JSX (inserir após a seção "Contato" e antes da seção "Condomínio"):
    ```jsx
    {/* Seção Cartões */}
    <div style={{ background: 'var(--color-surface)', borderRadius: 'var(--radius-card)', padding: 24 }}>
      <p style={{ fontSize: 12.5, fontWeight: 600, color: 'var(--color-text-sec)', marginBottom: 16 }}>Cartões</p>
      <SavedCardsList
        cards={savedCards}
        loading={loadingCards}
        error={cardError}
        mode="manage"
        onSetDefault={handleSetDefault}
        onRemove={(id) => handleRemovePress(savedCards.find(c => c.id === id)!)}
        removingId={removingCardId}
      />
      {!loadingCards && !cardError && savedCards.length > 0 && savedCards.length < 3 && (
        <p style={{ fontSize: 12.5, color: 'var(--color-text-ter)', textAlign: 'center', marginTop: 16 }}>
          Adicione cartões ao fazer uma compra na tela de Créditos.
        </p>
      )}
    </div>
    ```
