| API → OneSignal | Push best-effort — falha silenciosa; chave no env server-side |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-02-01 | Elevation of Privilege | /admin/settings, /admin/condominiums, /admin/combos | mitigate | Role check `request.user?.role !== 'ADMIN'` inline no controller de cada handler — retorna 403 antes de qualquer operação |
| T-07-02-02 | Tampering | PATCH /admin/settings/cutoff | mitigate | Zod regex `/^\d{2}:\d{2}$/` valida formato HH:MM antes de upsert; valor fora do padrão retorna 400 |
| T-07-02-03 | Tampering | POST/PATCH /admin/condominiums | mitigate | Zod parse do body antes de qualquer chamada ao banco — padrão existente do projeto |
| T-07-02-04 | Tampering | PATCH /admin/combos/:id/promotion | mitigate | Zod valida `{ active: boolean }` antes de operar; discountValue fixo (15%) no serviço — não configurável via request |
| T-07-02-05 | Information Disclosure | GET /admin/* sem autenticação | mitigate | preHandler: [fastify.authenticate] em todas as rotas — 401 sem JWT |
| T-07-SC | Tampering | npm installs | accept | Nenhuma instalação nova — apenas código TypeScript |
