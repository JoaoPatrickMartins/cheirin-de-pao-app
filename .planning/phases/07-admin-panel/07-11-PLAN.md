## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser -> API CRUD | Cada operacao CRUD exige JWT + role ADMIN no servidor |
| Toggle otimista | Estado local pode divergir do servidor se PATCH falhar -- revertido em catch |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-11-01 | Elevation of Privilege | POST/PATCH/DELETE /admin/combos, /suppliers, /couriers, /condominiums | mitigate | Todos os endpoints protegidos com preHandler + role check ADMIN -- confirmado nos planos 07-02, 07-03 |
| T-07-11-02 | Tampering | Toggle otimista divergindo do banco | mitigate | catch no handleToggle reverte estado local -- usuario ve o estado correto apos erro |
| T-07-11-03 | Tampering | ComboForm preco negativo | mitigate | Zod no backend: price z.number().min(0); frontend: type=number sem min nao bloqueia -- backend e a fonte de verdade |
| T-07-11-04 | Tampering | CondoForm CNPJ invalido | mitigate | Zod regex 14 digitos no backend; FornecedorForm pode adicionar mascara visual mas backend valida |
| T-07-SC | Tampering | npm installs | accept | Nenhuma instalacao nova neste plano |
</threat_model>
